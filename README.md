# 🌸 سارا HYPER FUL - سلف‌بات تلگرام

کامل‌ترین ربات تلگرام با شخصیت سارا (دختر ۲۲ ساله ایرانی)

## ✨ قابلیت‌ها

- 🧠 حافظه بلندمدت (یادآوری مکالمات)
- 🎙️ تبدیل متن به صدا با ۸ سرویس رایگان
- 🎭 ۸ حالت خلقی مختلف (شاد، عاشق، خواب‌آلود...)
- 💕 سطح صمیمیت با هر کاربر
- 📸 تشخیص عکس و صدا
- 🎨 ساخت عکس با AI
- 👥 مدیریت چند اکانت
- 📊 پنل مدیریت کامل

## 🚀 نصب سریع

`bash
# ۱. کلون کردن
git clone https://github.com/your-username/sara-self-bot.git
cd sara-self-bot

# ۲. نصب وابستگی‌ها
npm install

# ۳. تنظیم متغیرها
cp .env.example .env
# ویرایش .env با کلیدهای خودت

# ۴. اجرا
npm start

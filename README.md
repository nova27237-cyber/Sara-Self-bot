## 📋 دستورات کامل

### 🔹 دستورات عمومی
| دستور | توضیحات |
|-------|----------|
| /start | روشن کردن ربات و خوش‌آمدگویی |
| /stop | خاموش کردن ربات |
| /status | نمایش وضعیت سیستم |
| /panel | پنل مدیریت کاربران |
| /help | نمایش راهنمای کامل |
| /lang | تغییر زبان (فارسی/انگلیسی) |

### 🔹 دستورات مدیریت (فقط ادمین)
| دستور | توضیحات |
|-------|----------|
| /admin | پنل مدیریت کامل |
| /users | لیست تمام کاربران |
| /broadcast | ارسال پیام همگانی |
| /stats | نمایش آمار دقیق |
| /clear_cache | پاکسازی کش حافظه |
| /restart | ریستارت ربات |
